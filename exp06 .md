## Display empno, ename, deptno (department name using CASE instead of DECODE in MySQL)
```sql
SELECT EMPNO, ENAME,
CASE DEPTNO
    WHEN 10 THEN 'RESEARCH'
    WHEN 20 THEN 'ACCOUNTING'
    WHEN 30 THEN 'SALES'
    WHEN 40 THEN 'OPERATIONS'
END AS DEPARTMENT_NAME
FROM EMPLOYEE;
```
## DISPLAY AGE IN DAYS
```sql
SELECT DATEDIFF(CURDATE(), '2007-01-09') AS AGE_IN_DAYS;
```
## DISPLAY AGE IN MONTHS
```sql
SELECT TIMESTAMPDIFF(MONTH, '2007-01-09', CURDATE()) AS AGE_IN_MONTHS;
```
## DISPLAY CURRENT DATE FORMAT
```sql
SELECT DATE_FORMAT(CURDATE(), '%D %M %W %Y') AS FORMATTED_DATE;
```
## DISPLAY FORMATTED OUTPUT FOR EACH EMPLOYEE
```sql
SELECT CONCAT(ENAME,' has joined on ',
DATE_FORMAT(HIREDATE,'%W %D %M %Y')) AS DETAILS
FROM EMPLOYEE;
```
## SCOTT HAS JOINED THE COMPANY
```sql
SELECT CONCAT(ENAME,' has joined the company on ',
DATE_FORMAT(HIREDATE,'%W %D %M %Y')) AS DETAILS
FROM EMPLOYEE
WHERE ENAME='SCOTT';
```
## FIND NEAREST SATURDAY AFTER CURRENT DATE
```sql
SELECT DATE_ADD(CURDATE(),
INTERVAL (7 - DAYOFWEEK(CURDATE()) + 7) % 7 DAY) AS NEXT_SATURDAY;
```
## DISPLAY CURRENT TIME
```sql
SELECT CURTIME() AS CURRENT_TIME;
```
## DISPLAY DATE THREE MONTHS BEFORE CURRENT DATE
```sql
SELECT DATE_SUB(CURDATE(), INTERVAL 3 MONTH) AS THREE_MONTHS_BEFORE;
```
## EMPLOYEES WHO JOINED IN DECEMBER
```sql
SELECT * FROM EMPLOYEE
WHERE MONTH(HIREDATE) = 12;
```
## FIRST 2 CHARACTERS OF HIREDATE + LAST 2 OF SALARY
```sql
SELECT ENAME,
CONCAT(LEFT(HIREDATE,2), RIGHT(SAL,2)) AS RESULT
FROM EMPLOYEE;
```
## EMPLOYEES WHOSE 10% SALARY = YEAR OF JOINING
```sql
SELECT * FROM EMPLOYEE
WHERE (SAL * 0.10) = YEAR(HIREDATE);
```
## EMPLOYEES WHO JOINED BEFORE 15TH DAY OF MONTH
```sql
SELECT * FROM EMPLOYEE
WHERE DAY(HIREDATE) < 15;
```
## EMPLOYEES WHO JOINED BEFORE 15TH OF MONTH
```sql
SELECT * FROM EMPLOYEE
WHERE DAY(HIREDATE) < 15;
```
## EMPLOYEES WHOSE DEPTNO IS 10 OR 20
```sql
SELECT * FROM EMPLOYEE
WHERE DEPTNO IN (10,20);
```
